Etiquetas HTML5

Etiquetas para trabajar con texto

<p> </p>

Esta etiqueta representa un párrafo

  • Ejemplo:

    <p>Esto es un párrafo</p>

    Esto es un párrafo

<em> </em>

Esta etiqueta sirve para poner un texto en cursiva

  • Ejemplo:

    <em>Hola bebes</em>

    Hola bebes

<strong> </strong>

Esta etiqueta sirve para poner un texto en negrita

  • Ejemplo:

    <strong>Roat to 10</strong>

    Road to 10

<small></small>

Esta etiqueta sirve para poner un texto mas pequeño que el texto anterior

  • Ejemplo:

    El Jordi <small>poco listo</small>

    El Jordi poco listo

<s> </s>

Esta etiqueta sirve para poner un texto tachado

  • Ejemplo:

    <s>Hola Sergi</s>

    Hola Sergi

<h1-h6> </h1-h6>

Esta etiqueta representa un titulo o subtitulo

  • Ejemplo:

    <h2>Esto es un título</h2>

    Esto es un título

    <h4>Esto es un Subtítulo</h4>

    Esto es un Subtítulo

<u> </u>

Estilo de texto subrayado

  • Ejemplo:

    <u>Texto subrayado</u>

    Texto subrayado

<big> </big>

Estilo de texto grande

  • Ejemplo:

    <big>Texto grande</big>

    Texto grande

<abbr> </abbr>

Representa una abreviación

  • Ejemplo:

    <abbr>WWW</abbr>

    WWW

<data> </data>

Representa un valor de fecha y hora

  • Ejemplo:

    <data>12-01-2016 10:15</data>

    12-01-2016 10:15

Etiquetas para enlaces

<a> </a>

Esta etiqueta sirve para poner un hiperenlace

  • Ejemplo:

    <a>github
    </a>

    github

Etiquetas para ordenar contenido

<section> </section>

Remarca las selecciones de nuestro sitio web

<div> </div>

Define un bloque de contenido o de sección de página

<header> </header>

Es la cabecera de nuestra página web

<footer> </footer>

Es el pié de la página

Etiquetas para citas

<cite> </cite>

Esta etiqueta sirve para representar un titulo

  • Ejemplo:

    <cite>Celestina</cite>

    Celestina

Etiquetas para imagenes

<img> </img>

Esta etiqueta sirve para poner una imagen

  • Ejemplo:

    <img>src="img/RealMadrid.jpeg"</img>

Etiquetas para listas

<li> </li>

Esta etiqueta sirve para hacer una lista

  • Ejemplo:

    <li>Cereales</li>

    <li>Leche</li>

  • Cereales
  • Leche

    • <ol> </ol>

    Lista ordenada

    • Ejemplo:

      <ol>Cereales</ol>

      <ol>Leche</ol>

      Cereales
      Leche

    <ul> </ul>

    Lista no ordenada

    • Ejemplo:

      <ul>Cereales</ul>

      <ul>Leche</ul>

      Cereales
      Leche
