Estilos

Etiquetas

Una etiqueta o marca HTML es un código que se incluye en los archivos creados con el lenguaje HTML para estrucutar, añadir significado o formato al contenido de una página web.

ID

ID se utiliza para definir áreas específicas de una página (el header, el footer, un menú, etc)

Usaremos un ID si esa etiqueta es única, no debería haber dos etiquetas con el mismo ID en la misma página.

Para definir las propiedades de un ID antes del nombre ponemos el simbolo #

Clases

Se usa para definir estilos de tipo general (enlaces, listas, etc).

Usaremos class si es un estilo que repetiremos en diferentes etiquetas.

Para definir las propiedades de una clase antes del nombre ponemos el simbolo #
