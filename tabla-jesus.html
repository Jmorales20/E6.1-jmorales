Tabla CSS

Atributo Descripción Explicación Ejemplo
background-color Especifica el color de fondo de un elemento Como parámetro hay que poner el color en formato hex, rgb o rgba  background-color: #0B3B0B
background-image Inserta una imagen de fondo Como parámetros hay que poner url()  background-image: url()
background-repeat Establece cómo se repetirá una imagen de fondo Como parámetros hay que poner repeat, repeat-x, repeat-y, no-repeat, initial, inherit;  url de la imagen en cuestión
font-size Sirve para modificar el tamaño del texto Como parámetros utiliza medium, small, large, smaller, larger, % o length  font-size: medium
font-weight Define el grosor del texto Como parámetros utiliza normal, bold, lighter o number  font-weight: bold
font-family Define el tipo de fuente Como parámetros utiliza nombres de fuentes  font-family: "Comic Sans MS", cursive, sans-serif
text-align Define la posición del texto Como parámetros utiliza left, right, center o justify  text-align: center
text-transform Define el texto en mayusculas o minusculas Como parámetros utiliza none, capitalize, uppercase o lowercase  text-transform: capitalize
border Define los bordes Como parámetros utiliza Style, color, width, top, left, rigth o bottom  border-color: green
list-style-type Especifica el tipo de marcador de la lista de ítms en una lista Como parámetros utiliza circle, decimal, lower-roman, lower-alpha, none  list-style-type: none
text-decoration Añade decoraciones al texto Como parámetros utiliza none underline, overline o line-through  text-decoration: underline
padding Define el espaciado Como parametros utiliza bottom, left, top o right  padding-top: 5px;
margin Define el espaciado Como parámetros utiliza bottom, left, top o right  margin-bottom: auto;
